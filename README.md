# Alliance for OpenUSD (aousd)
The Alliance for OpenUSD (AOUSD) is a Linux Foundation project dedicated to promoting interoperability of 3D content through Universal Scene Description (OpenUSD). Founded by Pixar, Adobe, Apple, Autodesk, and NVIDIA, it standardizes 3D ecosystems for film, gaming, architecture, and industrial design.
